# Celeste

Tienda de ropa online para el mercado boliviano.

## Stack
- **Frontend**: Next.js 14 (SSR)
- **Backend**: Medusa.js v2
- **Database**: PostgreSQL
- **Payments**: Stripe / QR local
- **Deploy**: Vercel (frontend) + Railway (backend)

## Estructura
celeste-backend/   # Medusa.js v2
celeste-frontend/  # Next.js 14
